# AppTurnos

Repositorio inicial para la aplicación "AppTurnos".

Descripción

Este repositorio contendrá la aplicación AppTurnos. Aquí irán el código fuente, la documentación y los recursos relacionados.

Cómo clonar este repositorio

1. Instala Git en tu equipo (instrucciones abajo).
2. En la terminal o Git Bash ejecuta:

   git clone https://github.com/Josebalonso10/AppTurnos.git

3. Entra en la carpeta del proyecto:

   cd AppTurnos

Contribuye creando ramas nuevas y enviando Pull Requests.
